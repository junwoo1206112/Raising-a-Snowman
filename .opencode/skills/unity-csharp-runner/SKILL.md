# unity-csharp-runner

이 스킬은 `unity-cli exec` 명령어를 통해 유니티 런타임 환경에서 직접 코드를 실행할 때 사용합니다.

## 주요 기능
- **코드 실행:** 임의의 C# 코드를 에디터 컨텍스트에서 실행합니다.
- **데이터 조회:** 현재 씬의 오브젝트 상태나 시스템 변수를 실시간으로 확인합니다.
- **동적 생성:** 런타임 중에 게임 오브젝트를 생성하거나 컴포넌트를 수정합니다.

## 지침
- 복잡한 디버깅이 필요한 경우 `Debug.Log` 대신 `unity-cli exec "GameObject.Find(...)"`와 같은 방식으로 직접 조회하세요.
- 실행할 코드는 한 줄의 문자열로 전달하거나 파일 경로를 사용하세요.
- 실행 결과로 반환되는 값을 분석하여 다음 작업에 활용하세요.

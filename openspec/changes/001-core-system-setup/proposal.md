# Proposal: Core System Setup for "Raising a Snowman"

## 1. 개요
'슬레이어 키우기' 스타일의 방치형 RPG 시스템을 '눈사람' 테마로 구현합니다. 픽셀 아트 기반의 시각적 성장과 기하급수적인 수치 성장을 핵심 재미 요소로 합니다.

## 2. 핵심 메커니즘
- **전투**: 자동 횡스크롤 핵앤슬래시. 50마리 처치 후 보스 진입 (30초 제한).
- **성장**: 골드를 소모한 지수적 비용 기반 능력치 강화.
- **장비**: 4:1 합성을 통한 무기(나뭇가지/얼음송곳 등) 상위 등급 획득.
- **데이터**: 대용량 숫자 처리를 위한 단위 표기 시스템 (K, M, B, T...).

## 3. 차별화 요소 (눈사람 테마)
- **외형 변화**: 장비 및 승급 등급에 따라 눈사람의 크기와 장식(코, 눈, 팔, 모자)이 실시간으로 변화.
- **스킬 이펙트**: 눈보라, 고드름 낙하, 얼음 폭발 등 테마에 맞춘 시각 효과.

## 4. 기술적 전략
- **아키텍처**: VContainer를 이용한 의존성 주입.
- **최적화**: 몬스터 및 투사체 오브젝트 풀링.
- **데이터**: ScriptableObject를 이용한 몬스터/아이템 데이터베이스화.
